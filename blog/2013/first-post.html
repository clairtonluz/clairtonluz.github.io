Atalho para duplicar linha não funciona no gnome

31 January 2014

Para quem não sabe o atalh Ctrl + Alt + Up e Ctrl + Alt + Down serve para duplicar as linhas selecionada, e esse atalho não funciona no eclipse instalado no fedora 19 que vem com gnome. Para resolver isso basta abrir um terminal qualquer no seu linux e executar os seguntes comandos: 

$ gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-down "['']"
$ gsettings set org.gnome.desktop.wm.keybindings switch-to-workspace-up "['']"

pronto agora basta aproveitar seu atalho funcionando perfeitamente.
