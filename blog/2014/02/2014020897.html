Configurando servidor linux CentOS 6.5 para controle de banda.

08 February 2014

Este tutorial considera que você acabou de instalar o CentOS 6.5 e está com conexão a internet. primeira coisa a se fazer é atualizar o servidor. Instale o repositório RPMforge para conseguir acesso a um número maior de pacotes que iremos precisar, siga o seguinte tutorial:

Instalar repositório RPMforge para CentOS 6.x

Agora iremos configurar as interface de rede e o DHCP para que os clientes possam receber um ip automaticamente.

Caso suas intefaces de rede não esteja configurada siga o seguinte passo para configura-las.

OBS: A eth0 deve ser a intefaca onde chega o link sendo assim suponho que já esteja configurada.
Já a eth1 deve ter o ip estático pois este será o gateway da sua rede interna.
