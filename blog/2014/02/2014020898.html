Instalar repositório RPMforge para CentOS 6.x

08 February 2014

Você pode encontar a lista completa de pacotes RPMforge at http://packages.sw.be/ Baixe o versão compativel com sua arquitetura. Se você não sabe qual é sua arquitetura digite o seguinte comando para descobrir esta informação. i686 http://pkgs.repoforge.org/rpmforge-release/rpmforge-release-0.5.3-1.el6.rf.i686.rpm
x86_64 http://pkgs.repoforge.org/rpmforge-release/rpmforge-release-0.5.3-1.el6.rf.x86_64.rpm
Instale o GPG key com o seguintes comandos: Se você obter o seguinte erro é porque a chave já foi importada. Verifica o pacote que você baixou Instale o pacote Isso irá adicionar um arquivo de configuração de repository yum e importar as apropriadas GPG keys.
Então tente instalar alguma coisa como essa: Se você conseguiu instalar o htop entao está tudo pronto.


Fonte: http://wiki.centos.org/AdditionalResources/Repositories/RPMForge
