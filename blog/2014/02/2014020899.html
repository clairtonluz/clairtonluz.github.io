Configurar interface eth0, eth1 e etc em CentOS.

08 February 2014

Se sua interface de rede não está configurada então basta executar o seguinte comando e configurar você. É bem fácil e simples configurar por este comando veja abaixo: Escolha a opção Network configuration para configurar seu dispositivo de rede. Device configuration -> para configurar um dispositivo. Selecione New Device -> para adicionar seu novo dispositivo. Selecione Ethernet -> isso é pra definir qual o tipo de dispositivo que você que adicionar. preencha com os dados que desejar. lembrando que o campo Device é o nome que apararece no seu dispositivo quando você digita ifconfig -a.
